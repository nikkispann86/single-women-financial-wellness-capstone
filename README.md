# Single Women & Financial Wellness – Capstone Project

## Overview
This project analyzes the financial behavior, spending patterns, satisfaction levels, and estimated savings of single women. The goal was to uncover insights that could support financial wellness initiatives, customer engagement strategies, and product development.

## Tools Used
- Google Sheets – data cleaning, formatting, calculations
- Tableau – data visualization and dashboards

## Key Business Questions
- How does income relate to spending and savings among single women?
- What patterns exist between household size and financial satisfaction?
- How does employment status impact estimated savings?

## Key Insights
- A large portion of single women fall into lower estimated savings ranges, indicating potential financial vulnerability.
- Full-time employed women generally report higher income but also higher spending.
- Smaller household sizes are associated with higher financial satisfaction.

## Deliverables
- Cleaned dataset
- Tableau dashboards
- Written analysis
